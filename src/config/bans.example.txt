# --- Narrating comments (universal smell) ---
added_comment::^\s*//\s*(set|get|return|increment|loop over|call|create|update|delete|check if|initialize|now |then |this )::No narrating comments. Code says what it does; comments say WHY, rarely.
added_comment::^\s*#\s*(set|get|return|increment|loop over|call|create|update|check if|initialize|now |then )::No narrating comments. Code says what it does; comments say WHY, rarely.

# --- Types/interfaces inside class/service files (a common convention) ---
# Remove this line if your project co-locates types with classes.
type_in_class::^\s*(export\s+)?(interface|type)\s+\w+::Keep type/interface declarations out of class/service files; put them in their own types file.
